# DOCX 格式规范

受控技术文档排版标准。适用于所有由 doc-writer skill 生成的 DOCX 文件。
阶段 4 执行 pandoc 转换后，若需精确应用本规范，应运行 `scripts/apply_docx_format.py`。

---

## 1. 大标题 — Report Title (H1)

| 属性 | 值 |
|------|----|
| 中文字体 | 宋体 |
| 西文字体 | Times New Roman |
| 字号 | 小二 (18pt) |
| 颜色 | 纯黑 #000000 |
| 加粗 | 是 |
| 对齐 | 居中 |
| 行距 | 单倍行距 |
| 段后间距 | 1 行 (240 twips) |
| 段前间距 | 0 |
